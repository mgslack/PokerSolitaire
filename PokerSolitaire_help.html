Poker Solitaire - Help

Poker solitaire is a simple game in which a player will get dealt 25 cards to place in a 5 x 5 play area in which to get the best poker hands horizontally, diagonally and vertically. Need to lay out cards to get poker hands from a pair up to a royal straight flush.

Standard poker hands are scored as they are completed in the play area (have 5 in a row, column or diagonal) with the scoring using the American scoring rules. Score is based on the strength of the poker hand, where more points are given for the better hand. The scoring is as follows:

Points Hand
100 Royal Flush
75 Straight Flush
50 Four of a Kind
25 Full House
20 Flush
15 Straight
10 Three of a Kind
5 Two Pair
2 One Pair

Once the new game button is pressed, the first card is dealt to the player. Place the card in one of the 25 spots, and once placed, the next card is dealt. The play continues till all 25 cards have been dealt and placed in one of the slots.

The game is considered 'won', if the score of all of the hands is greater or equal to the winning score, which defaults to 150 points for this game. Normal poker solitaire winning score is usually 200 points, that can be changed to if wanted - or made easier (lower) if so chosen.

There is a built in 'hint' routine, but is fairly simplistic and an option to have the computer auto-play out the game. Note, if auto-played, the high score is not tracked. At this time, the auto-play basically plays out by getting a hint and playing the spot the hint routine selects.

Options:

There a couple of options that can be set for the Poker Solitaire game. To set, select options from the context menu (right click). The first option sets the card back image used for the card deck in the main form. The other option allows setting the winning score value (easy = 100 points, normal = 150 points and hard = 200 points). In addition, the window location is saved when the game is exited.
